# 위메프 프론트엔드 기술 면접 후기

가장 이상적인 프론트엔드 질문을 받았다.
면접관분들이 굉장히 젠틀하셨고, 질문도 다양하게 받았다.

## 1. 면접 질문 및 코멘트

- 자기소개
- 프로젝트 설명
- 장단점
- SPA에 대해서 설명
- Closure에 대해서 설명
- var, const, let에 대한 설명
- 렉시컬 스코프에대한 설명
- 브라우저 렌더링 과정에 대해 설명
- Promise에 대해 설명
- try-catch에 대한 설명
- Restful에 대한 설명
- SSO 연동에 대한 설명
- 프로젝트의 성능 최적화를 하는 방법에 대해 고민한 적이 있는지

## 2. 면접 후기
- 중간에 물 한 번만 마셔도 되는지 질문했었는데 다들 웃으셔서 좋았다.
- try-catch에 대한 설명을 하고, 거기에 대한 일화도 설명을 했었는데 이상적인 방향에 대해 설명을 들을 수 있었고 잘 대처 했다는 소리도 들을 수 있어서 인상깊었다.
