# 스마일게이트 프론트엔드 기술 면접 후기

면접을 봤던 팀은 AI팀으로, 면접관 네분과 면접을 보았다.
해당 팀의 프론트 개발자, 백엔드 개발자, 인사팀 한 분의 구성이었다.
이전 전형에서 인사담당자의 퇴사로 인해 담당자가 바뀌어서 연락에 혼선이 좀 있었지만, 무난하게 진행되었다.
면접관 분들은 친절하셨고, 질문도 다양하게 받았다.
면접은 화상으로 진행됐다.

## 1. 면접 질문 및 코멘트

- 자기소개
- 프로젝트에 대한 설명
- SSR과 CSR 장단점
- 브라우저 렌더링 과정
- 클로저 개념
- Vue script에 대해서
- Vue와 React의 차이점
- Nuxt를 써봤는지
- Next의 최신 업데이트 사항
- this에 대한 설명
- 렉시컬 스코프에 대해 설명
- 앱 라우터에 대해서
- 서버 관리는 어떻게 했는지
- 물리 서버에 대해서
- E2E 테스팅을 어떻게 했었는지
- 집과 회사의 거리

## 2. 면접 후기

- 면접관들이 질문을 많이 하셨고, 예상 시간보다도 더 길게 진행되었다.
- 이상적인 프론트엔드 면접이었던 것 같고, 다들 상냥해서 굉장히 인상적이었다.
- 개발자들이 Next의 버전 별 업데이트 사항들을 모두 알고 있어서 놀랐다. 나도 파악을 하고 있어야겠다는 생각이 들었다.
- Vue를 쓴지 좀 돼서, Vue script에 대한 부분을 잘 답하지 못해 아쉬웠다.
