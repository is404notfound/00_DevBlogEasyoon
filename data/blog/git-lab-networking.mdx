# Git Lab 소개와 네트워킹 데이

![](../static/images/git-lab-networking.png)

이 날은 선정님이 깃랩에서의 회사생활 소개와 함께, 네트워킹 데이를 진행하셨다.

### Git Lab은,,,

깃랩은 유명한 글로벌 서비스인 만큼 각 국의 사람들이 모여 일을 하기 때문에
구성원의 시차가 다양하고, 회의를 진행할 때에는 모두가 괜찮은 시간 대에 토의할 수 있는 날을 선정후 회의를 진행한다고 한다.

### 이외에도,,,

여러 재미있는 이야기들을 들을 수 있었는데,
어려운 개념에 대한 이야기는 유튜브 영상으로 제작해서 공유한다고 한다.
실제로 개념 설명에 대한 영상을 보여주시기도 했다.

### 원격으로 해외에서 일하는 경험,,,

원격으로 해외에서 일하는 경험은 정말 흥미로웠다.
어떻게 지원을 하게 되었는지, 어떤 과정을 거쳐서 깃랩에 들어가게 되었는지, 어떤 과정을 거쳐서 지금까지 일을 하고 있는지에 대한 이야기를 들을 수 있었다.
현재는 한국이 아닌 곳에서 거주를 하신다고 한다.

### 결론,,,

외국어 공부를 꾸준히 해야겠다는 생각이 들었다.
한 동안은 동기부여가 되어서 열심힝 외국어를 팠던 것 같다.
생각보다 세상에는 다양한 기회들이 있다고 느꼈다.
